# 安装prettier 代码格式化工具

 

#### 使用`npm install --save-dev --save-exact prettier`安装
- 使用`echo {}> .prettierrc.json` 在项目中生成
#### 安装husky、lint-staged（来对暂存的 git 文件运行校验）。`npm install --save-dev husky lint-staged`
- `npx husky install`  (安装前保证项目已有git地址)
- `npm set-script prepare "husky install"`
- `npx husky add .husky/pre-commit "npx lint-staged"`
- 安装完成项目中会生成一个.husky文件夹
- husky是常见的git hook工具，使用husky可以挂载Git钩子，当我们本地进行git commit或git push等操作前，能够执行其它一些操作，比如进行ESLint检查，如果不通过，就不允许commit或push。
- lint-staged来对暂存的 git 文件运行校验
